我們收集哪些資訊？

我們收集各種來自您或關於您的資訊，這些資訊視您使用的服務而定。

您的行為和所提供的資訊。

我們收集您在使用服務時所提供的內容和其他資訊，其我們收集哪些資訊？
我們收集各種來自您或關於您的資訊，這些資訊視您使用的服務而定。
您的行為和所提供的資訊。
我們收集您在使用服務時所提供的內容和其他資訊，其中包含註冊帳號、建立或分享內容，還有傳送訊息或與其他用戶交流。這可以是您提供的內容本身或其相關資訊，例如相片的拍攝地點或檔案的建立日期。此外，我們還會收集您如何使用服務的資訊，例如查看或互動的內容類型，以及您發佈動態的頻率和期間。

其他用戶的行為和所提供的資訊。

我們也收集其他用戶在使用服務時所提供的內容和資訊（包括您的相關資訊），例如分享有您在內的相片、傳送訊息給您，或者上傳、同步或匯入您的聯絡資料。

您的人脈網絡和關係鏈。

我們收集您連結的用戶和社團的資訊，以及您和他們如何互動的資訊，像是您最常交流的用戶或樂意與其分享事物的社團。
如果您從裝置上傳、同步或匯入聯絡資料（例如通訊錄），我們也會收集您提供的這些資訊。

付款方式的相關資訊。

如果您使用我們的服務從事購買行為或金融交易（像是您在 Facebook 上購買商品、在遊戲中購物，或者捐款），我們會收集該筆採購或交易的相關資訊。 其中包括您的付款資料，例如信用卡或簽帳金融卡號碼和其他卡片資料，以及其他帳號與驗證資料，還有帳單、出貨及聯絡詳情。

裝置資訊。

我們依據您授予的權限，收集您安裝或使用我們服務的電腦、手機或其他裝置本身或相關的資訊。
我們會彙整從不同裝置收集到的資訊，這有助於我們為您的各種裝置提供一致的服務。以下範例說明我們收集的裝置資訊種類：

作業系統、硬體版本、裝置設定、檔案和軟體名稱及類型、電池電量和訊號強度，以及裝置識別碼等屬性。

裝置位置，包含透過 GPS、藍牙或 WiFi 訊號等定位的特定地理位置。
連線資訊，例如行動電信業者或 ISP 名稱、瀏覽器類型、語言設定及時區、手機號碼和 IP 位址。

從使用我們服務的網站和應用程式取得的資訊。

如果第三方網站和應用程式使用我們的服務（例如提供「讚」按鈕或「Facebook 登入」，或是使用我們的衡量工具和廣告服務），則我們會收集您瀏覽或使用這些網站和應用程式時的資訊。
這些資訊包含您瀏覽的網站和應用程式、您在這些網站和應用程式中使用我們服務的情況，還有應用程式或網站開發人員或發佈者向您或我們提供的資訊。

第三方合作夥伴提供的資訊。

我們會收到由第三方合作夥伴所提供，在 Facebook 及其他網路空間中關於您和您的動態的資訊；例如我們與合作夥伴共同提供服務時，合作夥伴所提供的資訊，或是廣告商所提供的關於您的廣告體驗，或與其互動的資訊。

Facebook 企業。

依據 Facebook 持有或營運企業的條款和政策，我們會收到來自這些企業與您有關的資訊。深入瞭解這些企業和他們的隱私政策。

使用條款

使用條款（即權利與義務宣告）

本使用條款（即權利與義務宣告），以下簡稱「協議」、「條款」或「SRR」，係根據 Facebook 原則所訂定，是我們的服務條款，用以規範我們與用戶及其他與 Facebook 互動的人士的關係，同時也規範 Facebook 品牌、產品及服務（以上統稱「Facebook 服務」或「服務」）的關係。若使用或登入 Facebook 服務，即代表您同意本協議，且本協議將不時更新以符合以下第 13 條規定。除此之外，您亦可在本文件底部找到更多資訊，幫助您瞭解 Facebook 如何運作。由於 Facebook 提供的服務範圍相當廣泛，我們可能會要求您檢閱並接受適用於您與特定應用程式、產品或服務之互動關係的附加條款。若該補助條款與 SRR 有所牴觸，則您對於與補充條款相關之應用程式、產品或服務的使用均受補充條款規範，但約束範圍以不超出牴觸內容為限。

隱私設定

您的隱私對我們非常重要。我們設計了資料政策，以審慎詳細說明您可如何使用 Facebook 來與他人分享內容，以及我們可如何收集和使用您的內容及資訊。我們建議您詳閱資料政策，並使用此政策來協助您做出明智的決定。

分享您的內容和資料

您擁有您在 Facebook 發佈的所有內容和資料，您可透過隱私和應用程式設定管理您的分享內容方式。此外，智慧財產權所涵蓋的內容，如相片和影片（智財權內容），將根據您的隱私和應用程式設定，具體給予我們以下權限：您給予我們非獨有、可轉讓、可再授權、免版稅的全球授權，使用您發佈在 Facebook 或與 Facebook 聯的任何智財權內容（智財權授權）。當您刪除您的智財權內容或帳號，此智財權授權便宣告結束，除非您的內容已與他人分享而他們沒有刪除該內容。
當您刪除智財權內容，情況就如同在電腦上清理資源回收筒。然而，您應明白已刪除的內容可能會在合理時間內存有備份副本（但不會提供給他人使用）。
當您使用應用程式，該應用程式可能會要求授權存取您的內容和資訊、以及他人與您分享的內容及資訊。我們要求應用程式尊重您的隱私， 且您與該應用程式之間的協議將控制該應用程式對於您的內容和資訊將如何使用、儲存及轉讓。（欲暸解有關開放平台，包括您如何控制他人可以和應用程式分享您的哪些資訊，請詳閱我們的 資料政策 和 開放平台專頁。)
當您以「公開」設定發佈內容或資料時，即代表您允許所有人（包括 Facebook以外的人士）存取或使用該資料，並且將之與您關聯在一起（例如，您的名字和大頭貼照）。<
對於您所提供的意見或其他關於 Facebook 的建議，我們總是心存感激，但請明白我們沒有義務為使用這些意見或建議而提供您報酬（正如您沒有義務必須提供意見）。

安全性

我們盡力維持 Facebook 安全，但我們不能做出保證。要維持 Facebook 安全，我們需要您的協助，包括遵守以下承諾：
您不會在 Facebook 上發佈未經授權的商業通訊內容（如垃圾郵件）。
您不會收集用戶的內容或資料，或以其他方式在未經我們事先許可下，使用自動化方式登入 Facebook（如自動收成、機器人、網路蜘蛛或網路爬蟲等軟體）。
您不會在 Facebook 從事非法多層次傳銷，如金字塔式傳銷。
您不會上傳病毒或其他惡意程式碼。
您不會收集登入資料或意圖進入屬於他人的帳號。
您不會霸凌、恐嚇或騷擾任何用戶。
您不會發表以下內容：鼓吹仇恨、威脅、色情、煽動暴力或露點或血腥暴力傾向的內容。
在沒有適當的年齡限制下，您不會開發或經營包含與酒精相關、聯誼服務或其他成人內容（包括廣告）的第三方應用程式。
您不會使用 Facebook 進行任何非法、誤導、惡意或歧視的行為。
您不會進行任何會關閉、超載或損害 Facebook 正常運作和外觀的事情，如阻斷服務攻擊或干擾頁面顯示或其他 Facebook 功能等。
您不會協助或鼓勵任何違反本協議或政策的行為。

註冊和帳號保安

Facebook 用戶提供自己的真實姓名和資料，我們需要您的合作讓這種模式得以持續。以下是您對我們關於註冊和維護帳號安全的承諾：
您不會在 Facebook 提供任何虛假個人資料，或者未經允許為其他人建立帳號。
您不會建立一個以上的個人帳號。
如果我們停用您的帳號，您不會在未經我們許可的情況下建立另一個帳號。
您不會將個人的動態時報作為商業獲利的主要途徑，而會使用 Facebook 粉絲專頁作為此用途。
如果您未滿 13 歲，您不會使用 Facebook。
如果您已被定罪為性罪犯，您不會使用 Facebook。
您會提供正確的聯絡資料，並即時予以更新。
您不會分享您的密碼（或者，如為開發人員，您的開發商密鑰），讓他人登入您的帳號，或進行任何可能危及您帳號安全的行為。
在沒有得到我們事先的書面許可下，您不會轉讓您的帳號給任何人（包括您管理的任何粉絲專頁或應用程式）。
如果您為您的帳號或粉絲專頁選擇用戶名稱或類似的標識，在判斷為適當的情況之下（例如商標持有人投訴用戶名稱與用戶的真實姓名不符），我們將保留移除或收回的權利。

保護其他人的權利

我們尊重他人的權利，也希望您能比照辦理。
您不會在 Facebook 張貼侵犯、違反他人權利或違反法律的內容或任何行動。
如果我們認為您在 Facebook 張貼的任何內容或資料違反本協議或我們的政策，我們有權移除內容。
我們為您提供工具，協助您保護智慧財產權。若要瞭解更多，請瀏覽如何提出侵犯智慧財產權投訴聲明頁面。
如果您的內容因為侵犯他人的著作權而被我們刪除，而您認為我們的刪除有誤，我們會給您上訴機會。
如您重複侵犯他人的智慧財產權，我們會適時停用您的帳號。
在未經我們「品牌使用準則」的明確允許或取得我們的書面同意之前，您不得使用我們的著作權、商標或任何容易造成混淆的相似標記。
如果您收集用戶的相關資料，您必須：取得用戶同意，明確表示是您（而非 Facebook）收集他們的資料，並發佈隱私政策說明您收集的資料類型，以及您將如何使用這些資料。
您不會在 Facebook 上發佈任何人的身份證明文件或敏感財務資料。
您不會未經他人同意，便標註用戶或寄送電子郵件邀請給非用戶。Facebook 提供了社群檢舉工具，讓用戶可就標註功能提供意見。

在更改或取消您的手機號碼後，請於 48 小時內更新您的 Facebook 帳號資料，以確保您的訊息不會寄送到使用您舊號碼的用戶手中。
您給予同意及所有必要的權利，讓用戶使用裝置同步（包括通過應用程式）任何他們在 Facebook 上可以看見的資料。
